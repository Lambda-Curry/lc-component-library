# Icons

With `MDX` we can define a story for `Icons` right in the middle of our
markdown documentation. Out of the box we provide plenty of default icons that you can use throughout your app.
However, You can add or replace icons by wrapping your App with the Icon Registry component.

<Preview>
  <Story name='Icon'>
    <div className='icon-story'>
      <ul className='icon-story-grid'>
        {Object.keys(defaultIcons).map((iconName) => (
          <li key={iconName}>
            <Icon name={iconName} />
            <br />
            {iconName}
          </li>
        ))}
      </ul>
    </div>
  </Story>
</Preview>

# Icon Registry

You can add or replace icons by wrapping your App with the Icon Registry component.

```
import { ReactComponent as peaceSymbol } from './assets/hand-peace-light.svg';

export const App = ({ children }) => (
  <IconRegistry icons={{ peaceSymbol }}>
    {children}
  </IconRegistry>
```

<Preview>
  <Story name='Icon Registry'>
    <div className='icon-story'>
      <Icon name='peaceSymbol' />
    </div>
  </Story>
</Preview>
