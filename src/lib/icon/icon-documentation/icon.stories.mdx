# Icons

Out of the box we provide plenty of default icons that you can use throughout your app.
However, You can add or replace icons by wrapping your App with the Icon Registry component.

<Canvas>
  <Story

    }}

          {Object.keys(defaultIcons).map(iconName => (
            <li key={iconName}>

              <br />
              {iconName}
            </li>
          ))}
        </ul>
      </div>
    )}
  </Story>
  <ArgsTable of={Icon} />
</Canvas>

# Icon Registry

You can add or replace icons by wrapping your App with the Icon Registry component.

```tsx
import { ReactComponent as peaceSymbol } from './assets/hand-peace-light.svg';

export const App = ({ children }) => (
  <IconRegistry icons={{ peaceSymbol }}>
    {children}
  </IconRegistry>
```

Once the above code is implemented correctly, you should be able to use the Icon component to display your new custom icon.
